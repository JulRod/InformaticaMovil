# InformaticaMovil
Trabajo individual para la asignatura de Informática Móvil
Propiedad de Julio Rodríguez Soares

La aplicacion descarga el JSON de los bares de gijon de la pagina oficial de los datos libres del
Ayuntamiento de gijon.
Permite listar los favoritos del usuario o todos los bares, tanto en una lista como en la otra se tiene
la posibilidad de buscar tanto por nombre como por categoria.
La vista de detalle de un bar incluye nombre descripcion y foto y a partir de esa vista se podrá pasar 
a la ubicacion del mismo bar en un mapa de google maps.
